## linebot
### how to use 
- Step 0: Go to main path

- Step 1: Install Python Packages
pip install -r requirements.txt

- Start ngrok https server (default port:8787)
ngrok http 8787

- Step 2: Run main.py 
python main.py

### Directions
- A is first number  
B is second number  
A + B -->A plus B  
A - B -->A minus B  
A * B -->A multiply B  
A / B -->A divided by B
